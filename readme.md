Askora - Social Media Capstone Project Askora is a basic social media platform built as a capstone project. It provides a simple interface for users to view, create, and interact with posts, demonstrating foundational concepts of web development with only HTML and CSS. Features Static homepage with feed layout

Post creation form (ui only, non-functional)

Like and comment buttons (ui only, non-functional)

User profile card

Responsive, light aesthetic UI (white with blue or coral accents) Tech Stack HTML5

CSS3
